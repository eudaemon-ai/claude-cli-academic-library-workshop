About This Module

Academic reference work requires quickly assessing patron questions, identifying authoritative sources, and synthesizing information across multiple domains. In this module, you'll learn how Claude Code can help you draft polished reference responses faster—while keeping you in control of the final output.

You won't write any code. Instead, you'll practice giving Claude clear, context-rich prompts and refining its responses for your patrons.

What You'll Learn

  • How to orient Claude Code to your library's communication style using a context file
  • How to use /help to explore available commands and /clear to reset conversation context
  • How to reference files directly in prompts using the @ syntax
  • How to draft and refine reference responses for complex patron questions, including follow-ups
  • How to ask Claude to flag its own uncertain claims before you share output
  • How to iterate on Claude's output through audience reframing, transparency prompting, and constraints
  • How to use claude --print for quick one-off questions without opening a full session

Before You Begin

Make sure Claude Code is installed on your machine. Installation instructions for each platform:

  • macOS
  • Windows
  • Linux

Once installed, open a terminal and navigate to the library-context/ folder provided with this workshop. You'll run all exercises from that folder.
